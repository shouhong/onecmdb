ADDITIONAL DATA 

Here you can state if a certain Attribute in a certain 
CMDB Template shall get a specific, fixed value.
For example if you like to tag the data you now import you can have
an Attribute named ImportDate and set set it to "Imported on Nov 4 2009".

Click Add to define an addtional value that shall be set.
Select the correct Template and the Attribute that shall get the addtional value.
Enter the correct value.
