MDR - Overview 

Available MDR functions are:
Add       - Create a new MDR.
Configure - Configure an existing MDR.
Execute   - Execute eg. transform the selected MDR.
Open      - Show the result of the execution(transform) and commit data.
Refresh   - Refresh the mdr overview table.

A tutorial that describes how to use MDR is available
at www.onecmdb.org under documentation.
