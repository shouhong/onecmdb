When you are satisfied with the setup of this MDR Configuration you can 
save it using the Save-button. Then exit this configurator using the 
Finish-button.

After exiting this configurator you must select the configuration in the 
MDR table and press Execute.
Then press Start and wait for the execution to complete. 
Press Close to exit the Execution function.

Now select the configuration again and press Open.
You can now see what your execution has resulted in. 
Press "Commit to CMDB" if you like to import it to the CMDB.
